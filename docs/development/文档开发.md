# 文档开发

## 在线编辑

直接点击文档内底部的编辑此页快速跳转到GitHub编辑页面直接向本仓库提交你撰写的文档

## 本地开发

先决条件：

- nodejs(Version >= 16)
- ~~yarn(Version >= 1)~~（可选）

Clone 本仓库

在项目文件夹下运行：

```yarn
yarn install
yarn docs:dev
```

更新框架及插件(更新可能遇到兼容性问题)
