  - 若出现转存时路径留空转存无反应的情况, 请更新1.2.7版本

  - 加入了生成秒传的功能, 选择文件/文件夹后即可看到秒传生成按钮增加了跳转目录的功能, 若在秒传转存时有输入保存路径, 转存完成后可以看到跳转按钮
