# 关于脚本安装

此文档提供 Win/安卓 平台下的完整安装流程

\*其他平台和**无法安装脚本**的用户可尝试使用 [秒传网页版](https://rapidacg.gmgard.moe/)

\*教程仅为参考, 实际安装方案并不唯一
