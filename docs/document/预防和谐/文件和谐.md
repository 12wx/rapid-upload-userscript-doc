## 预防方法

 使用 **双层压缩**, 可规避在线解压导致的和谐, 详见 **文件压缩方法** 章节
2. **不使用** "分享链接" 分享文件, 可一定程度上规避举报导致的和谐(不开分享链接就没有举报按钮), 但不排除其他人使用秒传转存文件后再开分享链接.
